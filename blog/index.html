< 내가 공부하는 이유 - 나의 20살 시절 이야기 />

20살 시절, 막연히 컴공을 가야지, 하고 대학을 들어갔다. 이유는 단순했다. 그냥 고등학생때 컴퓨터와 스마트폰을 만지고 싶어도 ‘수능’ 이라는 입시제도 하에, 공부에 방해된다는 이유 하나로 맘대로 써볼수도 없었다. 그래서 컴공을 가면 이런 전자기기를 많이 만질 수 있을것 같아 컴공을 갔다. 하지만, 수능은 내가 원하는 대로 나와주지 않았고, 대학은 다 떨어지고 가장 하향으로 썼던 한 학교를 붙었다. 그렇게 시간은 흘러갔고, 오티도 끝나고 입학식을 하고, 이 학교의 학생이 되었다. 사실 학교가 마음에 안들긴 했지만, 나는 내 20살을 즐기고 싶었기에, 20살을 독서실 구석에서 보내기 싫었기에 그렇게 하지 않았다.

FR Migrate from Jekyll

France Move static content to static Jekyll has a rule that any directory not starting with _ will be copied as-is to the _site output. Hugo keeps all static content under static. You should therefore move it all there. With Jekyll, something that looked like ▾ <root>/ ▾ images/ logo.png should become ▾ <root>/ ▾ static/ ▾ images/ logo.png Additionally, you’ll want any files that should reside at the root (such as CNAME) to be moved to static.
